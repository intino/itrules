General Information

This document describes the Amidas Platform REST APIs, which allows users to access this platform programmatically.

The API accepts JSON content as input. The output is always in JSON (or empty, in the case of DELETE requests).

Check out the sections below for general information about all of APIs. To get right to it and dig into specific API, use the navigation on the left side of this site.

Authentication

This API supports token authentication. Only an account administrator can access or generate API key tokens.
